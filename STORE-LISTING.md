## Description (store "Detailed description" field)

Feed Beautifier gives you back control of your LinkedIn home feed. Choose a
reading layout that fits how you work, then switch off the parts you don't want
to see — all from a single popup, with changes applied instantly.

WHAT YOU GET

Three one-click layout presets:
• Zen — a distraction-free single column. Both side rails hidden so you focus on
  the posts and nothing else.
• Executive — a wider, information-dense reading column with the right rail
  hidden. Great for skimming quickly.
• Minimalist (Classic) — keeps the familiar layout but ready for your de-clutter
  toggles.

Standalone de-clutter toggles (mix and match with any preset):
• Hide promoted / sponsored posts
• Hide "engagement bait" and vanity widgets
• Hide the left sidebar
• Hide the right sidebar
• Silence the messaging overlay

WHY YOU'LL LIKE IT

• Instant — changes apply the moment you flip a switch, no page reload.
• Remembers you — your preset and toggles are saved locally and restored every
  visit.
• Lightweight — pure CSS restyling with a tiny content script. No tracking, no
  accounts, no external servers.
• Private by design — everything stays in your browser.

HOW TO USE

1. Pin the extension and open linkedin.com/feed.
2. Click the toolbar icon.
3. Pick a preset, then toggle anything you want to hide.

Source code and issues: https://github.com/jindalmt/feed-beautifier-linkedin

Feed Beautifier is an independent project and is not affiliated with, endorsed
by, or sponsored by LinkedIn.

## Permission justifications (Privacy practices tab)

storage:
Used to save the user's chosen layout preset and de-clutter toggle settings
locally in the browser so preferences persist between visits. No data leaves the
device.

activeTab / host access (www.linkedin.com/feed):
Required so the extension can restyle the LinkedIn feed page and apply the user's
selected layout. The extension only runs on linkedin.com/feed and does not read,
collect, or transmit page content.

Single purpose:
The extension has one purpose — let the user customize the visual layout and
reduce clutter on their own LinkedIn home feed.

Are you using remote code? No.

Data usage — this item does NOT:
• collect or use personally identifiable information
• collect health, financial, authentication, or location data
